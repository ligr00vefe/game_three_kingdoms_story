# 게임 기능 명세서 (GAME_DESIGN.md)

작성일: 2026-07-10 | 이 문서는 **게임이 어떻게 동작해야 하는지**를 기능별로 정의한다. 기술 구현 방법은 다루지 않는다 (그건 DEVELOPMENT_PLAN.md).
모든 수치는 초기값이며 플레이 테스트 후 조정한다.

---

## 1. 게임 개요

- **장르**: 2D 횡스크롤 액션 RPG (브라우저)
- **벤치마크**: 메이플스토리 — 조작감, 타격감, 사냥 루프, UI 배치를 참고한다. 원본 에셋·캐릭터는 사용하지 않고 스타일만 참고
- **세계관**: 삼국지. 역병으로 죽은 자들이 좀비가 되어 일어난 난세. 플레이어는 장수가 되어 좀비 무리를 토벌한다
- **핵심 재미 루프**: 사냥 → 경험치/아이템 획득 → 레벨업/장비 강화 → 더 강한 사냥터로

## 2. 주인공: 관우

- 붉은 얼굴, 긴 수염, 녹색 전포가 특징인 장수. 메이플 스타일의 2~3등신 비율로 표현
- **시작 무기: 청룡언월도** — 리치가 긴 대도. 휘두를 때 큰 호(弧)를 그리는 궤적이 시각적 정체성
- 시작 스탯(초기값): HP 100 / MP 50 / 공격력 10 / 이동속도 보통
- 향후 다른 장수(장비, 조운 등)를 선택 캐릭터로 추가할 수 있는 구조를 전제

## 3. 조작

| 입력 | 동작 |
|---|---|
| ←/→ | 좌우 이동 |
| Alt (또는 Space) | 점프 |
| Ctrl | 기본 공격 (청룡언월도 휘두르기) |
| Shift | 스킬 사용 |
| 더블탭 ←/→ (또는 별도 키) | 대쉬 |
| 점프 중 대쉬 키 | 점프 대쉬 (공중 돌진) |
| ↑/↓ (줄·사다리 근처) | 줄/사다리 타기 |
| ↓ + 점프 | 아래 발판으로 하강 |
| Z | 아이템 줍기 |
| I | 인벤토리 열기/닫기 |
| ↑ (포탈 위) | 포탈 이동 |

키 배치는 옵션에서 변경 가능하게 할 예정(후순위).

### 3.1 이동/점프
- 좌우 이동은 즉각 반응 (가속 없이 일정 속도, 메이플 감성)
- 점프는 1단 점프만. 점프 중 좌우 조작 가능
- 공중에서 공격 가능 (공중 휘두르기)

### 3.2 대쉬 / 점프 대쉬
- **대쉬**: 바라보는 방향으로 짧은 거리를 빠르게 이동. 쿨타임 1.5초(초기값). 대쉬 중 잔상 연출
- **점프 대쉬**: 점프 상태에서 대쉬 입력 시 공중에서 수평 돌진. 지상 대쉬보다 이동 거리 김. 착지 전 1회만 가능
- 대쉬 중에는 피격되지 않는다(무적 프레임) — 회피 수단으로서의 역할

### 3.3 줄/사다리
- 줄·사다리 구간에서 ↑/↓로 승강. 타는 동안 공격 불가
- 타는 중 점프로 이탈 가능

## 4. 전투

### 4.1 기본 공격 (청룡언월도 휘두르기)
- Ctrl 입력 시 전방을 크게 베는 모션. 연속 입력 시 모션이 이어짐
- **휘두르기 이펙트**: 칼날 궤적을 따라 초승달 모양 참격 이펙트 표시. 관우의 상징 연출이므로 타격감의 핵심
- 리치: 캐릭터 전방 약 1.5캐릭터 폭 (언월도의 긴 리치 반영). 한 번에 최대 3마리 동시 타격(초기값)
- 공격 중에는 이동 불가 (공중 공격 제외)

### 4.2 스킬
- Shift로 사용, MP 소모. 첫 스킬 1종으로 시작:
  - **청룡참(가칭)**: MP 15 소모. 전방 넓은 범위를 강하게 베어 기본 공격의 2.5배 데미지. 푸른 용 형상의 대형 참격 이펙트 + 화면에 짧은 타격 정지감(히트스톱). 쿨타임 5초
- 스킬은 레벨업으로 추가 습득하는 구조를 전제 (스킬창 UI는 후순위)

### 4.3 피격과 데미지 표시
- 몬스터 공격에 맞으면 관우가 뒤로 밀리며 짧은 경직 + 붉은 점멸. 피격 후 1초 무적
- 데미지 숫자는 타격 지점 위로 떠오르며 사라짐. 내가 준 데미지 / 받은 데미지 색상 구분
- 크리티컬(확률 5%, 1.5배)은 더 크고 화려한 숫자로 표시

### 4.4 데미지 계산 (기획 규칙)
- 기본 데미지 = 공격력 × 무기 배율 × 난수(0.9~1.1)
- 스킬 데미지 = 기본 데미지 × 스킬 배율
- 몬스터 방어력만큼 감산. 최소 데미지는 1 보장

## 5. 성장

### 5.1 경험치와 레벨업
- 몬스터 처치 시 경험치 획득. 경험치바가 가득 차면 레벨업
- **레벨업 연출**: 캐릭터 주변에 빛 기둥 이펙트 + "LEVEL UP!" 표시 (메이플 오마주). 이 순간 HP/MP 전량 회복
- 레벨업 시: 최대 HP +10, 최대 MP +5, 공격력 +2 (초기값, 자동 분배)
- 필요 경험치는 레벨마다 증가 (1→2레벨 100, 이후 레벨당 ×1.2)

### 5.2 HP / MP
- HP 0이 되면 사망: 회색 화면 + 부활 버튼 → 스테이지 시작 지점에서 부활 (사망 페널티는 초기엔 없음)
- HP/MP는 비전투 시 초당 소량 자연 회복. 물약(소비 아이템)으로 즉시 회복
- 화면 좌하단에 HP(빨강)/MP(파랑)/경험치(노랑) 바 상시 표시

## 6. 몬스터

### 6.1 테마
- 전 몬스터 공통 테마는 **좀비화된 삼국지 세력**. 스테이지가 진행될수록 강한 세력의 좀비가 등장

### 6.2 첫 몬스터: 황건당 좀비
- 누런 두건을 쓴 좀비 농민병. 느릿하게 걷다가 플레이어를 발견하면 다가와 할큄
- 행동 패턴: 배회(좌우 왕복) → 감지(플레이어 접근 시) → 추적 → 근접 공격
- 스탯(초기값): HP 30 / 공격력 5 / 경험치 15 / 이동 느림
- 처치 시: 쓰러지는 모션 후 소멸, 아이템 드랍 판정
- 드랍(초기값): 동전(상시), HP 물약(30%), 누런 두건 조각(10%, 추후 퀘스트 재료용)

### 6.3 리젠(재생성)
- 맵에 정해진 스폰 지역이 있고, 지역당 최대 마릿수 유지 (스테이지 1: 최대 8마리)
- 처치 후 5~10초 사이 랜덤 시점에 스폰 지역에서 재등장. 등장 시 땅에서 기어나오는 연출(좀비 테마 활용)

## 7. 스테이지 1: 초원

- **배경**: 넓은 초원. 하늘/구름(원경) → 먼 산 → 풀밭(근경)의 다층 배경이 시차를 두고 흐름
- **구성**: 가로로 긴 평지 + 낮은 언덕 발판 2~3개 + 사다리 1개 + 장애물(바위, 수레 잔해)
- **등장 몬스터**: 황건당 좀비만
- **NPC**: 촌장 1명 (맵 좌측 안전지대) — 말을 걸면 대화창으로 스테이지 안내
- **목표**: 자유 사냥. 좀비를 처치하며 레벨업과 아이템 획득 (퀘스트 시스템은 후순위)
- **클리어 개념 없음**: 사냥터형 맵. 추후 포탈로 다음 맵 연결

## 8. 아이템과 인벤토리

### 8.1 드랍과 줍기
- 몬스터 처치 시 아이템이 바닥에 떨어짐 (통통 튀는 연출)
- Z키로 줍기. 캐릭터와 겹친 아이템부터 획득. 드랍 후 60초 지나면 소멸
- 동전은 자동 획득 (줍기 키 불필요)

### 8.2 인벤토리
- I키로 열기. 격자형 슬롯(초기 24칸), 탭 구분: 장비 / 소비 / 기타
- 소비 아이템(물약)은 인벤토리에서 클릭 또는 단축키로 사용
- 장비는 착용 시 스탯 반영 (시작: 청룡언월도 고정. 장비 교체는 후순위)
- 인벤토리를 열어도 게임은 멈추지 않음 (메이플 방식)

### 8.3 아티팩트
- 스테이지에서 낮은 확률로 얻는 특수 장신구. 착용 시 고유 효과 부여
  - 예시(초기 1종): **적토의 편자** — 이동속도 +10%
- 아티팩트 전용 슬롯 1칸으로 시작, 추후 확장

## 9. NPC와 대화

- **NPC**: 맵의 지정 위치에 서 있는 비전투 캐릭터. 머리 위에 이름표 표시
- **대화창**: NPC 근처에서 ↑ 또는 클릭 시 화면 하단에 대화창 표시 — NPC 초상화 + 대사 텍스트 + (필요 시) 선택지 버튼. 대화 중에도 게임은 계속 진행
- **말풍선**: NPC나 몬스터 머리 위에 뜨는 짧은 텍스트 거품 (예: 촌장이 주기적으로 "좀비를 조심하게!"). 캐릭터를 따라다니며 몇 초 후 사라짐
- **공지사항 배너**: 화면 상단에 흐르거나 고정되는 시스템 공지 영역 (예: "황건당 좀비의 습격이 시작됐습니다!"). 서버에서 내려주는 텍스트를 표시

## 10. 지형 요소와 맵 구조 규칙

### 10.1 맵의 수직 구조 (레퍼런스: 메이플스토리 맵 분석)

- **바닥 규칙**: 캐릭터는 화면 최하단(bottom)에 직접 닿지 않는다. 캐릭터가 걷는 **보행로(바닥) 이미지가 있고, 그 아래에 이동 불가 장식 밴드(underFloor — 수면·연꽃·바위 등)**가 항상 존재한다
- **층 구분**: 맵은 1개 이상의 층(floor)으로 구성. 층은 발판 묶음으로 표현하며, 층 사이 연결은 반드시 아래 10.2의 이동 경로 유형 중 하나 이상을 배치한다
- **배경 레이어 구분(5층)**: ① 하늘/구름(스크롤 0.05) ② 먼 산/봉우리(0.2) ③ 성곽·나무 중경(0.5) ④ 보행로/지형(1.0) ⑤ 바닥 아래 장식 밴드(1.0). 레이어별 개별 이미지로 제작 (AI_IMAGE_PROMPTS.md 3~4장)

### 10.2 이동 경로 유형 (맵 설계 시 조합)

| 유형 | 동작 | 규칙 |
|---|---|---|
| **점프 이동 (분리형 발판)** | 떨어져 있는 발판 사이를 점프로 건넘 | 간격은 점프/점프대쉬 도달 범위 이내. 아래에서 점프로 올라탈 수 있고 ↓+점프로 내려감(원웨이) |
| **계단형 이동** | 계단을 걸어서 자동으로 오르내림 | 1단 = 폭 32px × 높이 16px. 점프 없이 걷기만으로 승강 |
| **경사형 이동** | 경사면을 걸어서 오르내림 | 경사도 약 26도(1:2). 물리는 계단형(잘게 쪼갠 스텝)으로 근사 |
| **줄/사다리 이동** | ↑/↓로 수직 이동 (3.3) | 사다리(ladder)와 줄(rope)은 시각만 다르고 조작 동일. 층 간격이 점프 높이를 넘으면 필수 배치 |
| **포탈** | 맵 간 이동. ↑로 사용 | 스테이지 1에는 잠김 상태 1개 |

- **장애물**: 바위·수레·황건적 바리케이드 등 통과 불가 오브젝트. 점프로 넘어야 함

### 10.3 스테이지 1 적용

- 지면선 아래 96px의 수면(연꽃) 장식 밴드 → 캐릭터 발밑에 항상 바닥 이미지가 보임
- 2층 구조: 지상층 + 발판층(분리형). 연결: 사다리 2개 + 계단 1개 + 점프 발판
- 배경: 하늘 → 먼 산 → 성곽 중경(삼국지 성 외곽) → 초원 보행로 → 수면 밴드

## 11. 추후 확장 (이번 범위 아님)

- **멀티플레이**: 같은 맵에서 다른 플레이어가 보이고 함께 사냥. 채팅 말풍선
- **음향**: 배경음악(스테이지별), 타격음, 스킬음, 레벨업 팡파레
- **퀘스트**: 촌장 NPC의 의뢰 (두건 조각 수집 등)
- **추가 장수/무기/스킬 트리, 상점, 강화 시스템**
